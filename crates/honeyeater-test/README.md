# honeyeater-test

Dev-only test helpers for the honeyeater DSP library: the seven tolerance assertion macros, a `.npy` reference-vector loader, and a scipy subprocess helper for live cross-validation.

This crate is not published to crates.io and is not intended for use outside honeyeater's own test suite. See `docs/policies.md` and `docs/roadmap.md` for the test methodology.
